# Prompts Index

This directory contains **reproducible prompt patterns**  
for AI-generated videos used in education and technical contexts.

The goal is **stability and correctness**, not creativity.

---

## Prompt Categories

### 1. Abstract / Mood

Used for:
- lecture introductions
- presentation openings
- setting atmosphere only

Files:
- `abstract_intro.md`
- `tech_mood.md`
- `research_lab.md`

Rules:
- no text
- no numbers
- no symbols
- short duration only

---

### 2. Image to Video

Used for:
- schematic diagrams
- conceptual figures
- static illustrations

Files:
- `image_to_video_basic.md`

Rules:
- structure must not change
- motion must be minimal
- discard if geometry is altered

---

## General Rules

- AI video is **context**, not content
- Accuracy comes from static images
- If unsure, do not animate

---

## Status

This prompt set is considered **stable**.
New prompts should be added only if:
- they introduce a new category, or
- they are required by a real use case
